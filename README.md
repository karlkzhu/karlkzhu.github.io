# Karl Zhu

Source for Karl Zhu's personal academic website at <https://karlkzhu.github.io>.

## Local development

Install the Ruby dependencies, then run Jekyll:

```sh
bundle install
bundle exec jekyll serve
```

The site is configured through `_config.yml`. Page content lives in `_pages`, and academic content is organized in `_publications`, `_talks`, `_teaching`, and `_portfolio`.
